Esta aplicación es una herramienta diseñada para practicar y memorizar frases en chino mandarín mediante un proceso guiado que combina:

- Traducción del chino al español  
- Escritura en Hanzi  
- Escritura en Pinyin (con números, convertido automáticamente a tonos)

El objetivo es ofrecer un entorno sencillo, rápido y accesible desde el móvil, funcionando incluso **sin conexión** gracias a su arquitectura PWA.
